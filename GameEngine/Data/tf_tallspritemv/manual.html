How to use these materials

Images included in the 'ex_characters' folder are additional materials not included in the standard materials.
Here, we will introduce how to use these materials.

Undeads

'Undead.png' is for skeletons in the upper row and for zombies in the lower row.
Also, from the left, adult men, adult women, children, and large people are arranged.

Skeletons and zombies can be used naked as they are, but I think it would be interesting to combine them with other characters as shown in this picture.
Such processing is possible with any graphic tool that has a layer function.

Succubus and Angel

The upper left of the 'Succubus.png' is the same as the lower right of 'Monster2.png', so to speak, a completed sample.
Others are parts to be combined.
It is assumed that it will be used by overlapping with a graphic tool with a layer function.
No incubus is provided.

The upper left of the 'Angel.png' is the same as the upper right of 'Monster2.png', so to speak, a completed sample.
Others are parts to be combined.
It is assumed that it will be used by overlapping with a graphic tool with a layer function.
The body is female, but I don't think angels are male or female.

This is an example of adding wings and angel rings to a normal character.

A succubus is a type of demon, so it might be interesting to use it for whispering between demon and angel.

Snakepeople

The top left of '!SnakePeople1.png' is an adult male body, and the right is an adult female.
The lower left is men's hair and clothes, but if you adjust the position, you can also use it for women.
The right side is the hair and clothes for women.

Let's make your favorite Snake People (Naga, Lamia, Gorgon) by combining various bodies.
It is assumed that it will be used by overlapping with a graphic tool with a layer function.

'!SnakePeople2.png' is an example combination, and of course you can use it as is.
On the bottom right are two kinds of wings adjusted for Snake People. It is suitable for women's height, but if you raise the position, it can also be used by men.

Be sure to add "!" to the file name when using it. If you don't, it will be displayed strangely on the top.

Statues

It's good that the character has a tall body, but the statues in RPG Maker MV have a small head, and I don't think they're modeled after the same species.
So I made some statues that were aligned with tall characters.

It is assumed to be placed in the 'characters' folder and set in the event, but you can also put it in the tileset and use it.
As it is, '$Statue.png' is 6x4 instead of 3x4, which is the character standard, so please delete unnecessary parts or copy it to an appropriate character set and use it.
Since it is oriented from top to bottom, left, right, and up, it is the same direction as a normal character, so you can specify the direction with [Set Movement Route].
From left to right: no pedestal, with pedestal, neck and no pedestal, with neckless pedestal
As for the swordsman, there are only right and left facing, and in the remaining parts, a beautiful pedestal, a slightly weathered pedestal, a broken pedestal, and a shadow are placed.
It is assumed that it will be used by overlapping with a graphic tool with a layer function.

'!StatueOption.png' is an option to be added to the statue (goddess statue), and it is assumed that it is placed in the charcter folder and set to the event, but please use it freely by processing it, such as combining it with the statue from the beginning.
On the left, 8 types of orbs blink with a [Stepping] animation.
The first row on the right is the orb that lost its light, cup, and pot, the second row is the cup with the liquid, the third row is the water falling from the pot up to the pedestal, the fourth row is the water falling from the pot up to the ground.
Rows 2 to 4 also support [Stepping] animation.

This is an example of orb usage.
There are several ways to create a display similar to this one, such as creating a character that combines a statue and an orb, or placing a statue in a tileset and placing only the orb in an event.

This is an example of how to use cups and pot.
Use it in the same way as the Orb, in a way that is convenient for you.

Maid

A 'Maid.png' is assumed that it will be used by overlapping with a graphic tool with a layer function.
In addition, there are also 5 pre-assembled characters, so please copy them to an appropriate character file using a graphic tool etc. and use them.

A 'Maid2.png' is an additional variation that also includes children and fairies.
There are also 6 pre-assembled ones.
Butler is not provided.

Men in robes

A 'menInRobes1.png' is great for making robed characters in temples.
It is assumed that it will be used by overlapping with a graphic tool with a layer function.
The upper row is an adult male, the lower row is an adult female, the first character from the left is the one with the hood removed, and the second character is the one with the hood.
For the rest, there are also 4 pre-assembled ones.

A 'menInRobes2.png' is 8 pre-assembled characters, again adult males in the upper row and adult females in the lower row.

Nurse

A 'Nurse1.png' is assumed that it will be used by overlapping with a graphic tool with a layer function.
The lower right is almost the same as the lower right of 'SF_People3.png', but the clothes have been adjusted so that it can be used as the state before transforming into the 6th character in 'SF_Enemy1.png'.

A 'Nurse2.png' is a color variation. The top and bottom are the same except for the hair color.
Male nurses are not available.

Others

Doors
